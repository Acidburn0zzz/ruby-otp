create an OTP instance. passphrase length must be >= 10 and <= 63. passphrase must only contains pure ascii characters (7 bits). seed should only contains alpha-numeric characters. seed must not contain spaces algo_str can be “md4”, “md5” (default), “sha1”, “sha256”, “sha384”, “sha512”.
